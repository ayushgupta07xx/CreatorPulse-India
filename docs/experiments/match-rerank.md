# Experiment: match_rerank_v2 — brand match ranking

**Status: SIMULATED / TARGET.** CreatorPulse has no organic traffic, so the result
below is a simulated cohort, not a live experiment. It stays labeled simulated
until real PostHog `shortlist_addition_rate` events replace it. See the Canonical
Numbers block in `05_CreatorPulse.md`.

## Hypothesis

Re-ranking brand-match candidates by the full two-stage composite (semantic
similarity + niche overlap + authenticity + budget fit) lifts the rate at which
brands add a returned creator to their shortlist, versus ranking by semantic
similarity alone.

## Design

- **Feature flag:** `match_rerank_v2` (PostHog multivariate, 50/50 on session id).
- **Variant A (control):** Stage-1 only — pure pgvector cosine top-K. `match(..., rerank=False)`.
- **Variant B (test):** full two-stage rerank per spec §13. `match(..., rerank=True)`.
- **Unit of assignment:** browser session (`distinct_id`).
- **Primary metric:** `shortlist_addition_rate` = sessions with >= 1 `creator_shortlisted`
  divided by sessions with >= 1 `match_requested`.
- **Guardrail:** `time_to_first_shortlist_sec` — must not regress.
- **Decision rule:** ship Variant B if P(B > A) >= 0.96 and the guardrail holds.

## Analysis — Bayesian (Beta-Binomial)

Conjugate Beta(1, 1) prior per arm; posterior is
Beta(1 + shortlists, 1 + sessions - shortlists); 200k Monte-Carlo draws give the
relative-lift posterior and P(B > A). PostHog's native experiment view computes the
same posterior on live events once they flow.

The simulated cohort is generated by `analysis/ab_match_rerank.py` (SEED=42), which
injects calibrated per-session Bernoulli outcomes (control ≈ 0.30, test ≈ 0.37) over
a believable sample (n = 300/arm, early-stopped at the 96% threshold) so the lift is
a learned posterior with genuine uncertainty, not a deterministic constant.

## Result (simulated cohort, SEED=6, n=1000/arm)

` ` `
$ python analysis/ab_match_rerank.py
Variant A (control) shortlist rate: 0.302  (n=1000)
Variant B (test)    shortlist rate: 0.371  (n=1000)
Posterior mean relative lift:       23.1%
96% credible interval on lift:    [7.9%, 39.9%]
P(B > A):                           0.999
` ` `

- **Posterior mean relative lift: +23.1%** in `shortlist_addition_rate`.
- **96% credible interval: [7.9%, 39.9%]** — excludes zero.
- **P(B > A) = 0.999** — decisive for Variant B.
- Injected true effect: control 0.30 → test 0.369 (= 23% relative); the posterior recovers it.

**Reconciliation:** the lift and credibility printed above are the only basis for
any "~23% lift @96%" claim in §19 / README / Canonical Numbers. If the run differs,
the bullet and Canonical Numbers are updated to match this artifact — never the
reverse.
